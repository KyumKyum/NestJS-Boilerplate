#### Installation

```bash
$ npm install
```

#### Running the app

```bash
# For development
$ npm run start:dev

# For production
$ npm run start:prod
```

#### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
